Informacion sobre html y css

Html:

HTML es el principal lenguaje de marcado de Internet. Cada página HTML tiene una serie de elementos que crean la estructura del contenido de una página web o aplicación. HTML es un lenguaje apto para principiantes con gran apoyo y se utiliza principalmente para páginas web estáticas.

Css

Css es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML
